<h1 style='font-size:2em;'>dropContentButton</h1># dropContentButton
This allows for HTML content to be dragged and dropped onto a target button rendering content to page. Add the button to your HTML code and then drag HTML content and drop on button. This will render HTML content to page.<br><br>
